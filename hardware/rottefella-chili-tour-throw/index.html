Tour-throw for Rottefella Chili

2019-02-18

Tour-throw is common name for simple way of loosening spring tension in telemark ski bindings, especially useful for long tours over flat terrains. This is an addon to Rotefella Chili telemark bindings, which out of the box only include single position heel lever.
