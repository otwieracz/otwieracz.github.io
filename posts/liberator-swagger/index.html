liberator-swagger

Jul 8, 2019 23:39 · 83 words · 1 minute read Software Clojure

Swagger become de-facto standard for REST API documentation, so it was obvious choice for describing liberator resources I was exposing in one of Emergynt projects. There is awesome metosin/spec-tools library enabling generation of Swagger definitions from Spec.

It seemed like good fit to merge liberator with spec-tools:

  • Documentation is written where inside resource definition, keeping code well organized and easy to understand.
  • Various handlers, like handle-not-found or handle-malformed can be generated automatically for meaningful error messages.

See liberator-swagger on GitHub for more info.
